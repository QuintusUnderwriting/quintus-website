# Quintus Underwriting Website

A professional Next.js App Router website draft for Quintus Underwriting, built with the supplied brand palette, premium editorial typography, and the current public site structure.

## Routes

- `/`
- `/about-us`
- `/products`
- `/property-owners`
- `/specialist-home-insurance`
- `/landowners-liability`
- `/bespoke-schemes`
- `/become-a-broker-partner`
- `/privacy-policy`
- `/terms-and-conditions`
- `/fca-notice`

## Local Development

```bash
npm install
npm run dev
```

## Vercel

The app is ready for Vercel deployment with the default Next.js build command:

```bash
npm run build
```
